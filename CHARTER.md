# Charter

`aoa-xda-connector` exists to turn public XDA thread/post evidence into local,
source-grounded packets an agent can query without touching private, write,
attachment, or account-gated routes.

The charter is not to mirror XDA. The charter is to preserve method, contracts,
fixtures, and evals so a local agent can install the connector, materialize a
small proof database, and later expand through bounded public seeds.

## Success Shape

- public-ready repo skeleton
- explicit source and storage policy
- no-network fixture proof
- reusable connector-family claim vocabulary
- evidence chains with source URLs, post IDs, and claim IDs
- local eval suites before live expansion
- heavy generated data excluded from Git
